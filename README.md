# Yousef Ismail Ahmed - Portfolio

A personal portfolio website created for **Assignment 2: Portfolio**.  
The website presents my professional identity, about section, experience, education, projects, courses and certifications, volunteering, and contact information.

## Live Demo

[View Portfolio Website](https://youssef-i-ahmed.github.io/Yousef-Ismail-Portfolio/)

## GitHub Repository

[View Source Code](https://github.com/Youssef-I-Ahmed/Yousef-Ismail-Portfolio)

## Assignment Sections Covered

- Personal Branding: full name, profile photo, and job title
- About Section: short introduction, specialization, key skills, and career objective
- Experience: roles, tasks, learning outcomes, and impact
- Education: university, major/specialization, and study period
- Projects: multiple practical projects with one major project highlighted
- Courses & Certifications: completed courses and certificates
- Volunteering: role description and contribution
- Contact Information: email, LinkedIn, and GitHub
- UI/UX & Presentation: clean layout, responsive design, and simple professional structure

## Main Features

- Responsive portfolio design
- Dark and light mode toggle
- Smooth scrolling navigation
- Animated section reveal effects
- Project cards with problem, solution, tools, and outcome
- Certificate preview modal
- Downloadable CV button
- Contact links for email, phone, LinkedIn, and GitHub

## Technologies Used

This project is built as a **static website** using:

- HTML5
- CSS3
- JavaScript
- Font Awesome icons
- Google Fonts
- GitHub Pages for deployment

> Note: This project does not use React, Vite, npm, or any JavaScript framework.

## How to Run Locally

1. Download or clone the repository:

```bash
git clone https://github.com/Youssef-I-Ahmed/Yousef-Ismail-Portfolio.git
```

2. Open the project folder.

3. Open `index.html` directly in the browser, or use the **Live Server** extension in VS Code.

No installation is required because the project is static HTML, CSS, and JavaScript.
